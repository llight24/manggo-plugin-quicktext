# manggo-plugin-quicktext

Manggo 快捷划词，会根据选择的文本弹出不同的功能



- 搜索。选择任意非空文本时显示“搜索”，点击后用默认浏览器搜索。默认搜索引擎为百度
  - 在“划词助手”的动作列表中点击该动作的编辑按钮，可切换为必应、Google 或自定义 URL。自定义 URL 可使用 `{query}` 作为选中文本的占位符，例如 `https://example.com/search?q={query}`；未使用占位符时，插件会自动添加 `q` 查询参数。

- 打开网址。选择 HTTP/HTTPS 网址时显示“打开网址”，点击后用系统默认浏览器打开。

- 运算。选择算术表达式时显示“计算并复制结果”。支持小数、括号及 `+`、`-`、`*`、`/`（也支持 `×`、`÷`）；结果会复制到剪贴板，并显示 Manggo 的复制通知。

- 查快递。选择含字母和数字的单段快递单号（长度 10–32）时显示“查询快递”。点击后会在默认浏览器打开 Bing 搜索。
- 打开文件夹。识别文本为文件路径后可打开文件夹
- 文件搜索，需搭配Everything使用，首次使用请在 `划词助手-文件搜索` 中配置 `Everything.exe` 路径

## 安装

运行：

```
npm run package
```

然后在 Manggo 的“插件管理”页面安装 `dist/manggo-plugin-quicktext.mplugin`。安装完成后，可在“划词助手”的动作列表右侧的编辑按钮选择搜索引擎，默认为百度，以及开启各项功能。
